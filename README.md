注意事项：

1.两个程序都需要下载，**缺一不可**

2.下载后**一定要先运行GUI.exe**，不要直接运行背诵程序，否则会因为Token缺失，导致无法获取翻译结果(程序对应行为：全选C)

3.**请不要修改FuckEnglish的名称**，否则图形界面会无法拉起背诵程序(水平太烂，只能关键字匹配)

4.GUI的名字可以随意修改
